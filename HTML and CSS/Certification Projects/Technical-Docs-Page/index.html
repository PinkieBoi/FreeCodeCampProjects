Overview

Python is a versatile high-level multi-platform programming language created by Guido van Rossum first released in 1991. It is popular among both beginners and seasoned developers due to its human readability and versitile nature.

Popular use cases include:

  • Server-side Web Development
  • Software Development
  • Machine Learning
  • Scripting

Installation instructions are available at python.org

Variables

Variables in python are must start with a letter or underscore and, other than underscores, consist of alphanumeric characters only.

Variables should be declared using snake casing - all lower case with words seperared by underscores:

> variable_name = "I am a variable"

Comments

Comments are used to explain code and make it easier for third parties to read and understand and are ignored by the interpreter
Comments generally begin with a '#' sign

# This code prints out the 'Hello World' string
>print("Hello, World !")
Hello World!

Multi line comments begin and end with tripple quotes (either double or single)

"""
This comment spans
many lines
"""
